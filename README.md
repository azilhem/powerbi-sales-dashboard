#  Sales Dashboard (Power BI)

---

## Project Overview

This project is an interactive **Sales Analysis Dashboard** built using **Power BI**.
It provides insights into revenue, customer behavior, product performance, and sales trends over time.

The dashboard is designed to help stakeholders quickly understand business performance and make data-driven decisions.

---

## Objectives

* Analyze overall **revenue performance**
* Track **sales trends over time**
* Identify **top-performing products**
* Compare performance across **product categories**
* Enable interactive exploration using filters

---

## Tools & Technologies

* **Power BI**
* **DAX (Data Analysis Expressions)**
* **Power Query (Data Cleaning & Transformation)**

---

## Key Features

### 🟣 KPI Indicators

* Total Customers
* Total Orders
* Total Revenue

### 🟣 Visualizations

* **Sales Over Time** (time series analysis)
* **Sales by Category**
* **Top 10 Products**

### 🟣 Interactivity

* Filters by:

  * Category
  * Region
* Dynamic updates across all visuals

---

## Key Insights

* **Technology** is the highest-performing category in terms of revenue
* Sales show **strong fluctuations over time**
* A small number of products contribute to a large portion of total revenue
* Regional and category filters reveal different performance patterns
